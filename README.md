Objetivos 
• Desenvolver um projeto interativo que utilize eventos CSS e HTML5 para criar 
uma experiência dinâmica e envolvente para o usuário. 

• Aplicar conceitos de programação orientada a eventos em um jogo simples. 
• Explorar transições e animações CSS para enriquecer a interação visual. 
Solução Digital

• erramenta utilizada: Visual Studio Code (VS Code) 
• Tecnologias: HTML, CSS e JavaScript 

• Execução: Jogo interativo de "Jokenpô" (pedra, papel, tesoura) com lógica 
implementada em JavaScript e interações capturadas por eventos HTML5. 
Procedimento / Atividade 



Atividade proposta: Criar um jogo simples de "Jokenpô" utilizando HTML, CSS e 
JavaScript. 

Etapas realizadas: 

1. Estrutura HTML com campos para escolher entre pedra, papel ou tesoura, área 
de resultado e botão de reinício. 
2. Estilização com CSS para criar animações simples e tornar o jogo visualmente 
atrativo. 
3. Implementação da lógica em JavaScript para determinar o vencedor e atualizar 
a interface dinamicamente.
